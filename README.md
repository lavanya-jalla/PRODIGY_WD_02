# Stopwatch Web Application

A modern, responsive, and feature-rich **Stopwatch Web Application** built using HTML, CSS, and JavaScript as part of my 1-month Web Development Internship at Prodigy InfoTech.

This project goes beyond basic start and stop functionality and focuses on UI/UX, responsiveness, animations, and real-world usability.



## Features

## Core Features
- Start / Pause / Reset stopwatch
- Millisecond-accurate timing
- Lap recording
- Keyboard shortcuts support
- Sound effects with toggle
- Fullscreen mode

## Advanced Features
- Smooth second-based tick sound (Web Audio API)
- Animated UI with glassmorphism design
- Scrollable lap list with animations
- Responsive layout for mobile, tablet & desktop
- Clean, modern UI with gradient background
- Accessibility-friendly tap targets

### UI / UX Enhancements
- Centered card with border & shadow
- Smooth hover and click animations
- Animated lap entry
- Logo integration with blended background
- Mobile-friendly design (media queries)



##Technologies Used

- **HTML5**
- **CSS3**
  - Flexbox
  - Media Queries
  - Animations & Transitions
  - Glassmorphism UI
- **JavaScript**
  - DOM manipulation
  - Web Audio API
  - Keyboard events
  - Timers (`Date.now()` based)



## Responsive Design

The application is fully responsive and optimized for:
- Mobile devices
- Tablets
- Laptops & desktops
